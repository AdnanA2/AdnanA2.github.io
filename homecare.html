HomeCare SaaS

Secure SaaS platform for managing Medicaid referrals, file uploads, and user roles.

Tech Stack

Java
Spring Boot
React
Docker
PostgreSQL

Screenshots

Key Features

  • Role-Based Access Control: Securely managed user roles (Admin, Case Worker, Viewer) with granular permissions.
  • Secure File Management: End-to-end encrypted file uploads, storage, and retrieval for sensitive patient documents.
  • Referral Management System: A complete workflow for creating, tracking, and managing Medicaid patient referrals.
  • Containerized Deployment: Fully containerized with Docker for consistent development, testing, and production environments.

Challenges & Solutions

  • Challenge: Ensuring HIPAA compliance for all data in transit and at rest.
    Solution: Implemented stringent encryption protocols, secure token-based authentication (JWT), and comprehensive audit logs.
  • Challenge: Building a dynamic and responsive user interface for complex forms and data tables.
    Solution: Leveraged React with a component-based architecture and state management to create a seamless and intuitive user experience.
